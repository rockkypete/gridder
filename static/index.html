About

Gridder® is a Self-changing DOM layout application; built with basic HTML5 markup, CSS3 rules and powered by Jquery.

Gridder was built to demonstrate DOM in action though, in a self changing approach. click the Change layout button on the right to the layout changing every 5mins. As a portfolio, gridder demonstrate great knowledge of frontEnd technlogies: HTML5 semantics, CSS3 flex and grid layout, as well as javascript ES6 standards such as atring literals, arrow functions, object destructuring and async functions (promise and await) to set time intervals for layout changes.
